## Portfolio Website

Personal portfolio website built with modern frontend technologies.

## Tech Stack
- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui

## Features
- Responsive design
- Dark theme
- Projects showcase
- Contact form
- Social links (GitHub, LinkedIn, LeetCode)

## Getting Started

### Prerequisites
- Node.js (v18+ recommended)

### Installation
```bash
git clone https://github.com/PallapatiAbhinairaj
cd your-repo
npm install
npm run dev
